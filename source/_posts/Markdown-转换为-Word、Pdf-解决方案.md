# 问题描述
&nbsp;&nbsp;写完的 markdown 文档，需要快速导出为其他格式。

# 解决方案
&nbsp;&nbsp;Windows 下采用插件方案，安装一个名为 `Writage` 的 word 插件，即可使用 word 打开 markdown 文件，可以此转换为其他格式的文档。
